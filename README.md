# 泰山職訓前端班時鐘作業
發揮創意美化你上課完成的時鐘，並試著加入更多功能  

## 注意
請盡量使用上課教的 GUI 工具或 git 指令繳交作業  
若使用 GitHub 網頁進行 commit 會依次數扣分

## 功能說明
- 火箭會隨時間往上飛，但實際上是背景在移動以營造火箭升空的效果
- 天空在每小時循環的20~40分鐘會逐漸變黑，模擬漸入太空的效果。(實際上是有一層overlay蓋在背景上，overlay初始設定為透明，後慢慢變黑)。
- 每小時循環的40分鐘後，會隨機生成星星。當秒數為3的倍數時，在隨機位置生成一顆。並透過CSS動畫實現閃爍的效果。
- 在每小時循環的最後兩秒(也就是59分58秒)，火箭會加速衝出視窗外(CSS動畫)。
- 完成每小時循環後，重置整個頁面。清除星星，並讓背景回到初始位置，overlay設為透明。
- 上方的數字時鐘與右上角的指針時鐘，時間是同步的。
- 指針時鐘上的波加曼是gif，並沒有甚麼厲害的。本來想做寶可夢主題的，但最後...至少放隻波加曼吧。
- 波加曼是我最喜歡的寶可夢!!
